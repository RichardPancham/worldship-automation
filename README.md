# UPS WorldShip Shipping Automation

A Python automation project that eliminates manual data entry in UPS WorldShip by reading customer orders from an Excel file and automatically pushing them into WorldShip's import system — so labels print with zero manual input.

---

## The Problem

In a typical UPS shipping workflow, staff manually re-enter customer order details (name, address, weight, service type) into WorldShip for every single shipment. This is:

- Slow — takes 2+ minutes per shipment
- Error-prone — wrong addresses, wrong service types
- Doesn't scale — more orders means more staff time

## The Solution

This script sits between your order intake and WorldShip. When a new order comes in:

1. Order details are logged in `orders.xlsx` with Status = `Pending`
2. Run `order_to_worldship.py`
3. Script writes all pending orders to WorldShip's official CSV import file
4. In WorldShip, use Keyed Import — type the order number and every field auto-populates
5. Hit Print — label comes out instantly

**No manual typing. No errors. Scales with any order volume.**

## How It Works

```
orders.xlsx          order_to_worldship.py        WorldShip
(Status=Pending)  →  reads + writes CSV       →  Keyed Import
                                                   auto-populates
                                                   label prints
```

This uses WorldShip's **official ODBC/CSV Keyed Import feature** — the same integration method used by large e-commerce companies connecting Shopify, ERPs, and order management systems to WorldShip.

## Getting Started

### Requirements
- Python 3.x
- Windows PC with UPS WorldShip installed

### Install
```bash
pip install openpyxl
```

### Setup
```bash
# Step 1 — create the Excel order template
python create_orders_excel.py

# Step 2 — open orders.xlsx and add your orders with Status = Pending

# Step 3 — export to WorldShip
python order_to_worldship.py
```

### WorldShip CSV location (default)
```
C:\ProgramData\UPS\WSTD\ImpExp\AcctPkgs\Sample Order Import 1\worldship.csv
```
Update `WORLDSHIP_CSV` in `order_to_worldship.py` if your path differs.
